# P2.7 Batch 7 - Reviewer Gate Routing Contract

Date: 2026-04-08
Scope task: B7.3

## Gate routing

| Gate | Paperclip reviewer agent |
|---|---|
| Script audit gate | `ReviewScriptAuditor` |
| Cost/infra gate | `ReviewCostInfra` |
| License/trademark gate | `ReviewLicense` |
| Phase-boundary gate | `ReviewPhaseBoundary` |

## Invocation contract

1. Batch closeout triggers all four reviewer agents.
2. Each reviewer agent run must emit a clear gate outcome token:
   - `SIGNED_OFF`
   - `APPROVED_WITH_NOTES`
   - `FINDINGS`
3. Batch is closable only when all four outcomes are recorded.
4. Findings require either:
   - remediation + rerun, or
   - explicit acceptance rationale in `docs/PHASE-1-PLAN.md`.

## Evidence contract

- Save reviewer outputs in `state/` artifacts per batch.
- Mirror gate outcomes in `docs/PHASE-1-PLAN.md` batch section.
- Sync canonical task status to Notion when MCP is available.
