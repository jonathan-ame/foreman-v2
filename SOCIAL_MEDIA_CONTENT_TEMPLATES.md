# Social Media Content Templates

**Purpose:** Consistent content frameworks for recurring social media themes and post types.

**Platform:** Primary focus on LinkedIn (format adaptable to Twitter/X).

**Usage:** Use templates as starting points, customize for specific context.

## 🎨 **VISUAL TEMPLATE GUIDELINES**

### **Carousel/Slide Structure:**
```
Slide 1: Hook/Problem Statement
- Bold headline
- Engaging visual
- Intriguing question

Slide 2-4: Content/Value
- Clear points with icons
- Data/statistics where relevant
- Progressive revelation

Slide 5: Solution/CTA
- How Foreman addresses
- Clear next step
- Link/reference

Slide 6: Engagement Prompt
- Question for comments
- Call to share/comment
- Hashtags
```

### **Image/Graphic Best Practices:**
- **Brand colors:** Primary blue (#2A5BD7), teal (#00B4D8), white/black
- **Typography:** Clean, readable fonts
- **Icons:** Simple line icons for concepts
- **Data visualization:** Clear charts/graphs
- **Consistency:** Similar style across all visuals

## 🔄 **CONTENT REPURPOSING FRAMEWORK**

### **Long-form to Social:**
```
Blog Post → Social Thread:
1. Extract key points (3-5)
2. Create thread connecting points
3. End with link to full article
4. Add engagement question

Case Study → Multiple Posts:
• Problem-focused post
• Solution-focused post  
• Results-focused post
• Quote/testimonial post
• Full story link post
```

### **Social to Social:**
```
LinkedIn → Twitter:
• Condense key insight
• Use thread format if multiple points
• Adjust hashtags for platform
• Platform-specific CTAs

High-performing Post → Variants:
• Test different headlines
• Try different visuals
• Post at different times
• Adjust call-to-action
```

## 📝 **COPYWRITING GUIDELINES**

### **Tone & Voice:**
- **Professional but approachable:** Business-focused but not corporate
- **Clear over clever:** Prioritize understanding over wordplay
- **Benefit-driven:** Focus on user outcomes
- **Authentic:** Share real experiences/learnings

### **Length Guidelines:**
- **LinkedIn posts:** 150-300 words optimal
- **Threads:** 3-7 tweets/posts
- **Questions:** Keep simple, answerable
- **CTAs:** Clear, action-oriented

### **Hashtag Strategy:**
- **Primary (1-2):** #AIagents #BusinessAutomation
- **Secondary (2-3):** #Solopreneur #Productivity #[Industry]
- **Brand (1):** #ForemanAI
- **Campaign (optional):** #[SeriesName]
